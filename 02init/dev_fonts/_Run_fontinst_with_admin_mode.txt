[폰트 확장자]
fon :일반 폰트 파일
ttf(True Type Font): 트루 타입 폰트
ttc(True Type Collection) : 트루 타입 콜렉션, ttf(ttf)를 묶어 놓은 것. 
                            
[윈도우에 폰트 설치 방법]
1. ttc, ttf 파일 마우스 우클릭 -> "설치"  선택
2. ttc ,ttf 파일 더블클릭 -> 폰트 뷰어에서 "설치" 선택 (ttc의 경우 ttf가 여럿이므로 "다음"으로 넘어가며 모두 설치해 줄것)
3. 윈도우는 "c:\windows\fonts" 에 모든 폰트가 설치 보관됨. ttf나 ttc 파일을 이 경로에 복사해 놓으면 1, 2번과 같이 설치한 것과 동일함
